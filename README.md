# Telegram Userbot + Control Bot

Управление своим аккаунтом через отдельного бота.

## Установка

1. Скопируй `.env.example` → `.env` и заполни данные
2. `pip install -r requirements.txt`

## Деплой на Render / Railway

- Подключи репозиторий
- Добавь Environment Variables (из `.env`)
- Build Command: `pip install -r requirements.txt`
- Start Command: `python main.py`

**Внимание**: Риск бана Telegram остаётся!
